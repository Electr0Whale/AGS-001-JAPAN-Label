# GBA SP (AGS-001) JPN Label Replica 🎮

*[English follows Chinese]*

本项目提供了一套用于打印日版 Game Boy Advance SP (AGS-001) 背部外壳标签的高精度复刻模板。

无论你是正在为你的 GBA SP 更换全新的外壳，还是想要修复磨损严重的旧标签，这个项目都能帮助你打印出近乎 1:1 还原原厂视觉效果的替换贴纸。

This project provides a highly accurate, printable template for the Japanese Game Boy Advance SP (AGS-001) back shell label. 

Whether you are reshelling your GBA SP or restoring a heavily worn original label, this project helps you print a nearly 1:1 replica of the authentic factory sticker.

---

## ✨ 特性 / Features

* **高精度排版 (High-Precision Layout)**：基于日版原机标签进行精确测量与临摹。 / Carefully measured and traced from original Japanese console labels.
* **完全可编辑 (Fully Editable)**：提供分层清晰的 `.psd` 源文件，你可以自由调整序列号或修改细节。 / Includes a well-organized `.psd` source file, allowing you to freely adjust the serial number or fine-tune details.
* **即打即用 (Ready to Print)**：提供导出的 `.png` 格式图片，无需 Photoshop 即可直接用于打印预览。 / Includes a high-resolution `.png` file for immediate use without Photoshop.
* **字体完美还原 (Accurate Fonts)**：附带了经过精挑细选的字体文件，确保文字的字重、间距与原版高度一致。 / Bundled with carefully selected font files to ensure the weight and spacing match the original perfectly.

## 📁 包含文件 / Included Files

项目仓库包含以下核心文件 (The repository contains the following core files):

* `AGS-001-JPN.psd`：Photoshop 源文件（包含完整图层与可编辑文本） / Photoshop source file (with intact layers and editable text).
* `AGS-001-JPN.png`：高清导出图像 / High-resolution export.
* **Fonts (字体文件夹)**：
    * `m+ 2c`
    * `Uni Ila Sundaram`
    * `OCR B Std`

## 🛠️ 使用指南 / Usage Instructions

### [中文]
1. **安装字体**：在打开 PSD 文件之前，请先将 `Fonts` 文件夹中的所有字体安装到你的操作系统中，以免出现字体缺失导致的排版错乱。
2. **编辑文件 (可选)**：使用 Adobe Photoshop 打开 `AGS-001-JPN.psd`。你可以双击文本图层，将条形码下方的序列号修改为你自己机器的真实序列号。
3. **打印建议**：
    * 为了达到最接近原厂的效果，建议使用**拉丝银或哑银 PET 不干胶贴纸**进行打印。
    * 表面建议覆**哑光膜**（消光膜），以增加耐磨性并还原原装标签的触感。

### [English]
1. **Install Fonts**: Before opening the PSD file, please install all fonts from the `Fonts` folder to your OS to prevent text rendering issues.
2. **Edit (Optional)**: Open `AGS-001-JPN.psd` in Adobe Photoshop. You can double-click the text layer to change the serial number below the barcode to match your actual console.
3. **Printing Recommendations**:
    * For the most authentic look, we recommend printing on **brushed silver or matte silver PET sticker paper**.
    * Apply a **matte laminate** to the surface to increase durability and replicate the original texture.
